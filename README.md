Database program ilk çalıştığında 'transcripts' klasörü üzerinden oluşuyor.

Transcripts klasörü içindeki her bir klasör YouTube'daki bir video Id'si. İçinde de o videoya ait transcript var.

---

İlk defa çalıştırırken:

npm run db:seed -> Database'i doldurur

---
